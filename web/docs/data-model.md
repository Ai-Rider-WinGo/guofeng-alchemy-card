# 数据模型草案

本目录为历史 Web 原型，数据模型以根目录 `docs/data-model.md` 为准。

关键配置结构：

- `card_definitions`
- `card_rarity`
- `card_types`
- `dynasty_tags`
- `gacha_pools`
- `gacha_pool_items`
- `gacha_pool_rotation`
- `gacha_logs`
- `user_cards`
- `user_card_shards`
- `user_collection_progress`
- `synthesis_recipes`
- `duplicate_conversion_rules`
- `weekly_collection_rewards`
- `user_reward_claim_logs`

现有 JSON 可继续作为 mock：

- `config/cards.json`
- `config/draw_pools.json`
- `config/merge_rules.json`
- `config/daily_limits.json`
