# 《国风炼金卡牌》MVP 范围

## 目标

第一版只验证核心玩法闭环：

抽卡 -> 获得历史卡 -> 同名升星 -> 人物关系合成 -> 解锁事件卡 -> 图鉴收集

暂不做完整 12 级经济、终极编号卡、支付、广告商业化和 2130 张母卡全量生产。

## 版本定位

MVP 名称：秦汉篇玩法验证版

目标平台：先做可交互原型，再接入抖音小游戏能力。

建议实现方式：

- 玩法原型：Taro React / React Web 均可
- 正式游戏表现：Cocos Creator 3.x
- 后端：先用本地 JSON 或轻量 Node.js API，确认玩法后再上 NestJS + PostgreSQL

## MVP 必做功能

1. 抽卡
   - 每日基础池
   - 单抽
   - 十连抽
   - 新卡/重复卡提示

2. 库存
   - 按卡牌 ID 聚合数量
   - 记录最高星级
   - 支持同名卡消耗升星

3. 合成
   - 两张指定卡合成一张更高级卡
   - 低级合成 100% 成功
   - 展示合成说明，解释历史关系

4. 图鉴
   - 展示已解锁卡牌
   - 展示未解锁卡牌占位
   - 可查看卡牌详情、故事、标签、合成线索

5. 新手路线
   - 围绕楚汉战争建立一条可跑通路线：
   - 刘邦 + 纪信 -> 荥阳脱困
   - 项羽 + 章邯 -> 巨鹿之战
   - 荥阳脱困 + 鸿门宴 -> 楚汉相争
   - 楚汉相争 + 垓下之围 -> 大汉开国

## MVP 暂不做

- 12 级终极编号卡真实发行
- 支付与订单
- 广告激励
- 分享奖励结算
- 后台管理系统
- 完整防沉迷/实名链路
- 全量 AI 卡图生产
- 高级合成失败率

## 成功标准

- 用户能在 3 分钟内理解：抽卡、重复卡、升星、合成、图鉴。
- 用户能在 10 分钟内完成至少 1 条历史事件合成线。
- 每次合成都能让用户知道“为什么这两张卡能合成”。
- 低级卡不是废料，而是可读的历史知识点。
