# 秦汉 MVP 卡牌文字内容

## 使用原则

卡牌图片不直接生成大段文字。卡名、稀有度、短句、历史简介、知识点、合成提示由配置和 UI 叠加，避免 AI 图片中文字乱码。

文字风格：

- 简洁
- 有历史感
- 不玄幻
- 不神怪化人物
- 适合卡面、卡册和详情页复用

## 字段说明

- 卡名：卡面主标题
- 类型：人物 / 地点 / 事件 / 阶段事件
- 稀有度：Common / Uncommon / Rare / Epic / Legendary
- 卡面短句：卡面展示的一句话
- 详情简介：卡牌详情页正文
- 知识点：图鉴学习点
- 合成提示：合成页或卡牌详情页提示
- 生成画面关键词：给 ComfyUI prompt 使用
