玩法规则

国风炼金 · 历史知识卡牌对战

🎯 核心玩法

收集不同历史板块的卡牌，通过 炼金合成 将低级卡升级为高级卡。同一历史的卡牌可以合成更高等级的终极卡。

📊 等级体系

壹 Lv1 贰 Lv2 叁 Lv3 肆 Lv4 伍 Lv5 Lv6-12

等级越高越稀有，合成需要消耗多张同级卡 + 时间/道具

🏛️ 六大板块

秦汉 三国 两晋 隋唐 宋元 明清

每板块覆盖该时期的人物、地点、事件、势力、人物关系、阶段事件

🃏 卡牌类型

人物 地点 事件 势力 人物关系 阶段事件

不同类型的卡有不同技能效果，组合搭配形成策略

🔄 炼金合成

2张同等级 + 1张同板块 = 1张高一级

消耗配方：3×Lv1 → Lv2，2×Lv2 + 1×Lv1 → Lv3，具体规则见 merge_rules.json

📅 每日/每周

每日抽卡池：上限N张（见 daily_limits.json）

每周收集奖励：集齐特定组合得额外卡（见 weekly_collection_rewards.json）

重复卡转换：3张重复 → 随机同等级（见 duplicate_conversion_rules.json）

📐 编号规则

XX-?-NNNN-LNN

XX=板块代码 ?=类型 NNNN=序号 LNN=等级

例：SG-P-0002-L02 = 三国·人物·0002·等级贰
