## 一、产品概述

「六朝炼金录」是一款面向抖音小程序的国风卡牌收集合成游戏。玩家收集秦汉、三国、唐、宋、明、春秋战国六个朝代的历史卡牌，通过合成升级，最终炼成 12 张至宝级卡牌，兑换含 1g 黄金的实体编号卡牌。

**核心驱动：** 集换式收藏 + 文化沉浸 + 社交竞技  
**变现模式：** 免费游玩 + 激励广告 + 品牌联名  
**终极钩子：** 实体黄金卡牌（12 款 × 100 张 = 1200 张）

## 三、核心玩法循环

### 3.1 日循环（5-10 分钟）

```
签到 → 抽卡(免费+广告) → 合成(通用/配方) → 完成任务 → 社交(交换/排行/展示)
```

### 3.2 周循环（核心节奏）

- **每周一 00:00：朝代轮换** — 新朝代 UP 卡池开放，上周朝代降为基础池
- **每周一 00:00：排行榜结算** — 上周排行奖励发放，新周排行开启
- **每周一 00:00：周任务刷新** — 周收集目标、合成挑战重置
- **周三：限时小活动** — 主题 mini-game
- **周末：双倍福利** — 碎片掉落翻倍、合成经验加成
- **6 周一循环**，之后重新轮转

### 3.3 月度大事件

- **月末实体卡窗口** — 符合条件的玩家申请兑换实体黄金卡
- **月度品牌联名上线** — 新联名卡池 + 品牌任务链
- **月度大师榜** — 跨朝代综合排行，授予"炼金宗师"称号

## 四、合成系统

### 4.1 三条路径

| 路径 | 机制 | 确定性 | 适用等级 |
|------|------|--------|----------|
| 通用合成 | 任意 2 张同等级同朝代 → 随机升 1 级 | 随机 | 全部等级 |
| 配方炼制 | 稀有配方 + 指定 3-4 张材料卡 → 指定目标卡 | 100% | Lv4-11 |
| 碎片直兑 | 纯碎片兑换指定卡 | 100% | 全部等级 |

### 4.2 通用合成成功率

| 等级段 | 成功率 | 失败返还 |
|--------|--------|----------|
| Lv1→Lv5 | 100% | — |
| Lv6→Lv8 | 80% | 返还 1 张原卡 |
| Lv9→Lv11 | 60% | 返还碎片 |
| Lv11→Lv12 | 不可 | — |

### 4.3 配方系统

- **配方是稀有消耗品**，通过里程碑、限时活动、惊喜掉落获得
- **配方 + 指定 3-4 张材料卡 = 保底目标卡**
- 每朝约 10-15 条配方
- 配方分级：稀有(Lv5-7) → 史诗(Lv8-10) → 传说(Lv11)

### 4.4 Lv12 唯一合成规则

- **Lv12 无配方、无碎片直兑、无通用合成**
- **唯一路径：收集 6 张指定 Lv11 神品卡 → 合成 1 张 Lv12 至宝卡**
- 6 张 Lv11 跨越 6 个朝代，每朝贡献 1 张
- 合成 Lv12 即获得实体卡兑换资格，进入混合兑换池

## 五、社交系统

### 5.1 好友交换

| 规则 | 限制 |
|------|------|
| 可交换等级 | Lv1-5 |
| 每日赠送上限 | 3 张 |
| 每日接收上限 | 5 张 |
| 需要关系 | 互相关注好友 |
| 碎片赠送 | 每日 10 碎片 |

- **心愿单**：每位玩家设 3 张心愿卡，好友可见并一键赠送
- **拉新奖励**：邀请→碎片(100)→配方→限定边框，三级阶梯

### 5.2 排行榜

- 收藏大师榜（总收集数）
- 炼金大师榜（合成次数）
- 朝代专家榜（当前周朝代收集度）
- 月度总榜（综合加权）
- 每周一结算，奖励称号、头像框、碎片、配方

### 5.3 炼金阁（个人展示）

- 自选 6 张卡牌展示
- 称号墙、成就展示
- Lv10+ 卡金色流光特效，Lv12 全屏特效
- 访客点赞（双方获碎片）、访客记录
- 一键生成截图分享到抖音动态

## 六、经济系统与变现

### 6.1 广告收入（主力）

| 广告类型 | 场景 | 频次 |
|----------|------|------|
| 激励视频 | 额外抽卡 +5 次/天 | 主动触发 |
| 激励视频 | 合成成功率 +20% | 主动触发 |
| 激励视频 | 碎片掉落翻倍 | 主动触发 |
| Banner/插屏 | 页面切换 | 被动展示 |

### 6.2 品牌联名（增量）

- 月度品牌合作，定制 3-5 张联名卡牌
- 限时联名卡池 + 品牌任务链
- 收费模式：月费制或分成制

### 6.3 实体卡经济

- Lv9-11 非金卡由代工厂批量生产，单张成本极低
- Lv12 金卡含 1g 黄金，按实时金价 + 制卡物流费用计算
- 13,800 张实体卡分 2-3 年逐步发放
- 实体卡整体是获客成本 + 传播投资，由广告和联名收入覆盖
- 13,800 张实体卡 = 13,800 次潜在晒单，自然流量价值远超成本

### 6.4 核心原则

- **免费可玩到底** — 广告是加速器，不是必需品
- **黄金锚定价值** — 实体含金量给虚拟收集真实价值
- **长线运营** — 周轮换 + 月事件 + 联名滚动更新

## 七、关键运营指标

| 指标 | 健康值 | 说明 |
|------|--------|------|
| DAU | 1 万+ | 广告变现基础门槛 |
| 人均广告次数 | 5-8 次/天 | 激励视频为主 |
| 次日留存 | >40% | 签到+任务+收集惯性 |
| 7 日留存 | >20% | 周朝代轮换拉回 |
| 邀请转化率 | >15% | 好友交换驱动裂变 |
| 品牌联名单价 | ¥3-10 万/月 | 随 DAU 增长上调 |

## 八、附录：与现有 MVP 的关系

当前 MVP（12 张卡，楚汉争霸，Lv1-5）可作为「六朝炼金录」的**秦汉朝代种子内容**进行扩展。现有系统（抽卡、合成、碎片、签到、任务）均可在新框架下复用和增强。
